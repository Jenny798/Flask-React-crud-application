#  Job Tracker Application

A full-stack web application to help users efficiently track and manage their job applications in one place. It includes authentication, job CRUD operations, a dashboard UI, and an AI chatbot assistant powered by Groq API.

---

## Features

-  User Authentication (Login / Register)
-  Add, Edit, Delete Job Applications
-  Dashboard to view all applied jobs
-  Track job status (Applied / Interview / Rejected / Selected)
-  AI Chatbot Assistant (Groq API)
-  Responsive UI using Bootstrap
-  Real-time API integration

---
